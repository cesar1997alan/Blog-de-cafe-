# Blog de Café

Proyecto de un blog temático sobre café desarrollado con HTML y CSS, enfocado en aplicar **buenas prácticas de rendimiento web y optimización** para mejorar la experiencia del usuario.

## Descripción
Este proyecto consiste en la creación de un blog informativo sobre café, donde además del diseño y la estructura del sitio, se implementan diversas técnicas para optimizar el rendimiento, accesibilidad y carga de recursos.

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript (básico)
- Modernizr

## Optimización y rendimiento
En este proyecto se aplicaron las siguientes mejoras de performance:

- Lazy loading para imágenes
- Uso de imágenes en formato WebP
- Implementación de `preload` y `prefetch`
- Etiquetas meta optimizadas para SEO
- Análisis de rendimiento con Lighthouse
- Detección de compatibilidad mediante Modernizr

##  Estructura del proyecto
- Página principal del blog
- Entradas informativas
- Secciones de contenido organizadas
- Diseño responsive adaptable a distintos dispositivos
